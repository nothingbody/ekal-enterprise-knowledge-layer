隐私政策

最后更新：2026 年 3 月 17 日

1. 概述

知枢（以下简称"本产品"）是一款支持私有部署的 RAG 智能知识平台。我们深知数据安全和隐私保护的重要性，本政策说明我们如何收集、使用和保护您的信息。

2. 数据存储

桌面版：所有数据（文档、向量库、对话记录、用户信息）均存储在您的本地设备上，不会上传至第三方服务器。

服务器版：数据存储在您自行部署的服务器上，由您完全掌控。

云端服务（可选）：若您启用了云端同步功能，以下信息会传输至中心服务器：

  • 设备标识符和软件版本
  • 聚合用量统计（知识库数量、文档数量、对话数量等，不含具体内容）
  • 操作日志摘要（用于审计和问题排查）

3. 第三方服务

本产品可能与以下第三方服务交互，这些服务由您自行配置：

  • LLM 模型服务（如 OpenAI、Anthropic、本地 Ollama 等）— 对话内容将发送至您配置的模型 API
  • Embedding 服务 — 文档文本将发送至您配置的向量化 API
  • Google Fonts — 页面加载时请求字体文件

我们建议您仔细阅读所选第三方服务的隐私政策。

4. 数据安全

我们采取以下措施保护您的数据：

  • API 密钥使用 AES 加密存储
  • 支持双因素认证 (2FA)
  • 操作日志记录所有关键操作
  • 支持数据备份和恢复
  • CSP、CSRF 等 Web 安全防护

5. Cookie

本产品使用浏览器 localStorage 存储登录令牌和用户偏好，不使用第三方跟踪 Cookie。

6. 儿童隐私

本产品非面向 14 岁以下儿童设计。我们不会有意收集儿童的个人信息。

7. 政策更新

我们可能不时更新本隐私政策。更新后的政策将在本页面发布，并更新"最后更新"日期。

8. 联系我们

如有任何隐私相关问题，请通过 support@zhishu.ai 与我们联系。
