定价常见问题

知枢软件本身收费吗？

不收费。知枢软件完全免费下载和使用。唯一的费用是 AI 模型的 API 调用费用，如果你使用本地模型（如 Ollama），则完全免费。

免费试用额度用完后怎么办？

试用额度用完后，你可以配置自己的 API Key（如 OpenAI、Anthropic 等）继续免费使用，或选择平台托管模式按用量付费。

什么是"自带 Key"模式？

你在知枢中配置自己的 AI 模型 API Key（如 OpenAI API Key），所有 AI 调用费用直接由你的账户与 API 提供商结算，知枢不额外收费。

支持哪些 AI 模型提供商？

支持所有 OpenAI 兼容接口，包括 OpenAI、Anthropic Claude、DeepSeek、通义千问、智谱 GLM 等。也支持通过 Ollama 运行 Llama、Qwen 等本地开源模型。
